## Features

### Aimbot *(new in this fork)*
| Setting | Default | Description |
|---|---|---|
| Active | `off` | Master toggle |
| Hold Key | `Left Alt` | Aim assist only while held |
| FOV | `10┬░` | Target cone тАФ enemies outside are ignored |
| Smoothing | `5.0` | 1 = instant snap ┬╖ 20 = very soft interpolation |
| Aim Bone | `Head` | Head / Neck / Upper Body |
| Only Enemy | `on` | Never aim at teammates |
| Only Visible | `on` | Trace check before locking on |
| TriggerBot | `off` | Auto-fire when crosshair overlaps target |
| Trigger FOV | `2┬░` | How precisely on-target before firing |

> Angle writes go through **`CL_Bypass::SetViewAngles`** тАФ the same protobuf CRC-spoof path used by the base, so subtick integrity is maintained.

---

### Visuals (ESP)
- **Player bounding boxes** тАФ 4 styles: Box ┬╖ Outline Box ┬╖ Coal Box ┬╖ Outline Coal Box  
- **Bone ESP** тАФ skeleton overlay on all players  
- **Glow** тАФ per-team colour via `Hook_DrawGlow`  
- **Visibility-aware colouring** тАФ colour shifts when enemy is traced  
- Independent **enemy / teammate** toggles  

---

### Anti-Detection & Bypass
- **`move_crc` bypass** тАФ all angle and attack writes are routed through the protobuf-patching path (`CL_Bypass`) so the server-side CRC check passes cleanly  
- **CS2 subtick input spoofing** тАФ `AddSubtickMoveStep` injects attack inputs at the correct `when` value within the subtick window  
- **Blackbone manual-map injection** тАФ bypasses standard DLL load path; injector with database in `Andromeda-Injector/`  

---

### Developer Infrastructure
- **Crash log** тАФ on exception, dumps a crash offset you can resolve in x64dbg: load the DLL, hit `Home`, subtract `0x1000` from the base, add the crash offset  
- **Schema dumper** тАФ set `DUMP_SCHEMA_ALL_OFFSET 1` in `Common/Include/Config.hpp` to dump all CS2 schema offsets  
- **Protobuf message parser** тАФ `Hook_ParseMessage.cpp` shows a live sound-position ESP example  
- **JSON config** тАФ `CSettingsJson` load/save via RapidJSON; config files drop next to the DLL  
- **`DEV_LOG`** тАФ `DEV_LOG("msg\n")` zero-overhead debug logging  
- **FW1 + FreeType font rendering** тАФ all fonts work in-game (see `CAndromedaClient.cpp`, `CVisual.cpp`)
